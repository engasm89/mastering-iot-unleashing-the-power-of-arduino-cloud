## Connections

- Analog temperature sensor (e.g., LM35): `VCC -> 5V`, `GND -> GND`, `OUT -> A0`.

## Diagram (ASCII)

```
 LM35           Arduino UNO
 +---+          +-----------+
 |OUT|----A0----| A0        |
 |VCC|----5V----| 5V        |
 |GND|----GND---| GND       |
 +---+          +-----------+
```

## Notes

- Calibrate the analog-to-Celsius conversion based on your sensor.
- For Arduino IoT Cloud, replace the serial prints with network client updates.
